**Open-source cheminformatics libraries for JavaScript**

Isomorphic TypeScript packages for chemistry, crystallography, and molecular data — works in Node.js and browsers.

---
- **[crystallography.io](https://github.com/chemistry/crystallography.io)** - Crystal Structure Search
- **[chemical-libraries](https://github.com/chemistry/chemical-libraries)** - Common Chemical Libraries
- **[molpad](https://github.com/chemistry/molpad)** - Simple Molecule Editor
- **[crystalview](https://github.com/chemistry/crystalview)** - Crystal Structure Visualization
