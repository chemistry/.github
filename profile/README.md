**Open-source cheminformatics libraries for JavaScript**

Isomorphic TypeScript packages for chemistry, crystallography, and molecular data — works in Node.js and browsers.

---
- **[@chemistry/common](https://github.com/chemistry/common)** - Common types and interfaces
- **[@chemistry/elements](https://github.com/chemistry/elements)** - Periodic table data
- **[@chemistry/formula](https://github.com/chemistry/formula)** - Chemical formula parser
- **[@chemistry/math](https://github.com/chemistry/math)** - Linear algebra for crystallography
- **[@chemistry/molecule](https://github.com/chemistry/molecule)** - Molecule data model
- **[@chemistry/space-groups](https://github.com/chemistry/space-groups)** - Crystallographic space groups
---
